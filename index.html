HTML Cheatsheet
  1. HTML Boiler Plate : This is the basic boiler plate of HTML 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. Heading Tag : There are basically six types of heading tags ranging from h1, h2, h3, h4, h5 and h6 with h1 being the largest and h6 the smallest 
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
    <h4>Heading 4</h4>
    <h5>Heading 5</h5>
    <h6>Heading 6</h6>
  3. Paragraph Tag 
    <p>This is a Paragraph</p>
  4. Anchor Tag 
    <a href="">This is a anchor tag</a>
  5. Handling Images in HTML : Here src is the field where you have to enter the URL 
    <img src="" alt="">
